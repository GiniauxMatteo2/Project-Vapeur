# Vapeur - A Dynamic Web Application for Managing Video Game Collections
 Vapeur is a dynamic web application designed to manage a collection of video games. Featuring a system for creating, editing, and viewing games, genres, and publishers, it provides an intuitive and user-friendly interface.  Main Features: Add, edit, and delete video games. Filter games by genre or publisher. Highlight games on the homepage. Manage genres and publishers.
